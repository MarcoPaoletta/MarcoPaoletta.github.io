ABOUT ME

Mi name is Marco Paoletta

Video game developer

I am 15 years old and I live in Buenos Aires city, Argentina. I was born on February 27, 2007.

I create my own video games in which I spend most of my time without omitting my high school studies.

My passion is to develop casual games with a simple but addictive gameplay for the player.

Most of my time is dedicated to programming since I consider it is the future of the world and I enjoy it very much.

I try to improve myself day by day, project after project and I am constantly learning something new.

MY JOURNEY AS A PROGRAMMER

Python

I started using Python in January 2021 at the age of 13 because I was curious about how a video game is made. I did few projects using Python because I felt limited and that I could give more of myself

GDScript

Therefore, I started learning Godot Engine, whose language, GDScript, has similar syntax to Python. I worked with Godot for about a year and I learned how to use it efficiently making a lot of awesome projects

HTML - CSS - JS

As I was having more and more projects, I learned the basics to create the Front-end part of web pages using HTML, CSS and JavaScript. This knowledge was applied, for example, in this portfolio website

C#

After I knew the basics of web development, I began to investigate about other game engines. So, I discovered Unity and its C# language: highly demanded and used for famous video games today
